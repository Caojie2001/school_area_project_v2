================================================================================
与数据库字段的对应关系
================================================================================

calculation_history 表字段映射：

学生数据：
  full_time_undergraduate     ← 全日制本科生
  full_time_specialist        ← 全日制专科生
  full_time_master            ← 全日制硕士生
  full_time_doctor            ← 全日制博士生
  full_time_total             ← 全日制学生总数
  international_undergraduate ← 留学生本科生
  international_master        ← 留学生硕士生
  international_doctor        ← 留学生博士生
  international_total         ← 留学生总数
  total_students              ← 学生总人数

教学及辅助用房：
  teaching_area_current               ← 教学及辅助用房_现状
  teaching_area_preliminary           ← 教学及辅助用房_拟建成前期
  teaching_area_under_construction    ← 教学及辅助用房_拟建成在建
  teaching_area_total                 ← 教学及辅助用房_汇总

办公用房：
  office_area_current                 ← 办公用房_现状
  office_area_preliminary             ← 办公用房_拟建成前期
  office_area_under_construction      ← 办公用房_拟建成在建
  office_area_total                   ← 办公用房_汇总

生活配套用房：
  total_living_area_current           ← 生活配套用房_现状
  total_living_area_preliminary       ← 生活配套用房_拟建成前期
  total_living_area_under_construction ← 生活配套用房_拟建成在建
  total_living_area_total             ← 生活配套用房_汇总

学生宿舍：
  dormitory_area_current              ← 学生宿舍_现状
  dormitory_area_preliminary          ← 学生宿舍_拟建成前期
  dormitory_area_under_construction   ← 学生宿舍_拟建成在建
  dormitory_area_total                ← 学生宿舍_汇总

后勤辅助用房：
  logistics_area_current              ← 后勤辅助用房_现状
  logistics_area_preliminary          ← 后勤辅助用房_拟建成前期
  logistics_area_under_construction   ← 后勤辅助用房_拟建成在建
  logistics_area_total                ← 后勤辅助用房_汇总

建筑总面积：
  current_building_area               ← 建筑总面积_现状

  under_construction_building_area    ← 建筑总面积_拟建成在建
  total_building_area                 ← 建筑总面积_汇总


================================================================================
测试建议
================================================================================

1. 基础数据完整性测试
   - 使用工作表"综合测试数据"中的完整数据
   - 验证所有字段都能正确保存和读取

2. 三阶段面积计算测试
   - 验证现状 + 前期 + 在建 = 汇总
   - 验证缺口计算时是否正确使用汇总值

3. 学生数据汇总测试
   - 验证本科+专科+硕士+博士 = 全日制总数
   - 验证全日制总数 + 留学生总数 = 学生总人数

4. 不同院校类型标准测试
   - 使用相同学生数，不同院校类型
   - 验证应配面积是否符合各类型标准

5. 边界值测试
   - 使用极小值（如100人）和极大值（如50000人）
   - 验证系统稳定性

6. 特殊补助测试
   - 添加多项特殊补助
   - 验证总缺口计算准确性
