【系统简介】

高校建筑面积缺口测算系统用于计算和分析高校建筑面积的规划需求与现状差距,
支持10种院校类型、5种用房类型的自动化测算,为高校基建规划提供科学依据。


【用户角色】

┌────────────────────────────────────────────────────┐
│ 角色       │ 权限范围                                │
├────────────────────────────────────────────────────┤
│ 管理员     │ 用户管理、标准配置、全部数据            │
│ 基建中心   │ 查看所有学校数据、统计分析(只读)        │
│ 学校用户   │ 本校数据录入和查看                      │
└────────────────────────────────────────────────────┘

================================================================================
                          一、管理员操作
================================================================================

1.1 用户管理
────────────────────────────────────────────────────────

【新建用户】
① 点击"用户管理" → "新建用户"
② 填写:用户名、姓名、密码、角色(管理员/基建中心/学校)
③ 角色为"学校"时需选择所属学校
④ 点击"确认"

【编辑用户】
① 在用户列表找到目标用户 → 点击"编辑"
② 可修改:姓名、角色、所属学校、状态
③ 点击"保存"

【重置密码】
① 点击用户的"重置密码"按钮
② 输入新密码 → 点击"确认"
③ 通知用户新密码,要求首次登录后修改

【删除用户】
① 点击用户的"删除"按钮 → 确认删除
⚠️ 删除操作不可恢复


1.2 测算标准配置
────────────────────────────────────────────────────────

【查看标准】
点击"测算标准配置",切换"基础面积标准"和"补贴面积标准"标签查看

【编辑基础标准】
① 找到需要修改的标准项 → 点击"编辑"
② 修改标准值(平方米/生)→ 点击"保存"
⚠️ 修改后立即生效,影响后续所有测算

【编辑补贴标准】
① 在"补贴面积标准"标签找到目标项 → 点击"编辑"
② 修改补贴值 → 点击"保存"


================================================================================
                        二、基建中心操作
================================================================================

2.1 面积管理
────────────────────────────────────────────────────────

【查看学校面积】
① 点击"面积管理"
② 选择学校、年份
③ 查看现有面积、拟建成面积、特殊补助


2.2 规划学生数管理
────────────────────────────────────────────────────────

【查看规划学生数】
① 点击"学生规划管理"
② 选择学校
③ 查看该校各年度、各口径的规划学生数记录

【新增记录】
① 点击"新增规划学生数"
② 填写:规划年度、测算口径、各类学生人数
③ 点击"保存"


2.3 统计分析
────────────────────────────────────────────────────────

【总体概览】
① 点击"统计分析"
② 查看:学校总数、测算记录数、总缺口面积、平均缺口比例
③ 可设置年份、院校类型、测算口径等筛选条件

【院校类型分析】
点击"院校类型分析"标签,查看各类型院校的缺口对比

【年度趋势】
点击"年度趋势"标签,查看多年度数据变化趋势

【批量导出】
① 点击"批量导出"
② 选择导出范围(全部/指定年份/指定院校/指定学校)
③ 点击"开始导出" → 下载ZIP文件


2.4 历史记录查询
────────────────────────────────────────────────────────

【查询记录】
① 点击"历史测算记录"或"数据管理"
② 使用筛选条件:学校、年份、测算口径、提交人
③ 点击"查询"

【批量下载】
① 勾选需要下载的记录
② 点击"批量下载" → 下载ZIP压缩包
⚠️ 单次最多50条记录


================================================================================
                         三、学校用户操作
================================================================================

3.1 数据录入(完整流程)
────────────────────────────────────────────────────────

① 点击"数据录入"
② 选择基准年份、规划年份、测算口径
③ 填写规划学生数:
   - 全日制本科生、专科生、硕士生、博士生
   - 留学生本科、硕士、博士
④ 填写现状面积:
   - 选择数据来源(自填/高校基础表/SEC数据)
   - 填写5种用房类型的现有面积
⑤ 填写拟建成面积:
   - 前期面积
   - 在建(含竣工)面积
⑥ 添加特殊补助(可选):
   - 点击"添加补助项目"
   - 输入项目名称和补助面积
⑦ 查看自动计算的测算结果
⑧ 点击"保存"
⑨ 点击"下载Excel报告"获取测算报告

【5种用房类型】
• 教学及辅助用房
• 图书馆
• 体育用房
• 行政办公用房
• 生活福利及其他用房


3.2 面积管理
────────────────────────────────────────────────────────

【维护现有面积】
① 点击"面积管理" → 选择年份
② 选择数据来源
③ 填写各用房类型面积 → 点击"保存"

【维护拟建成面积】
在"拟建成面积"区域填写前期面积和在建面积 → 点击"保存"

【管理特殊补助】
① 点击"添加补助项目"
② 输入项目名称、补助面积 → 确认
③ 可编辑或删除已有项目


3.3 学生规划管理
────────────────────────────────────────────────────────

【新增规划】
① 点击"学生规划管理" → "新增规划学生数"
② 填写规划年度、测算口径、各类学生人数
③ 点击"保存"

【编辑规划】
① 在列表中找到记录 → 点击"编辑"
② 修改数据 → 点击"保存"


3.4 查看测算结果
────────────────────────────────────────────────────────

【测算结果包含】
• 学生数据汇总(总数、当量)
• 各类用房测算(7个维度):
  - 现有面积、规划面积、在建面积
  - 汇总面积、应配面积
  - 缺口面积(不含特殊补助)
  - 缺口面积(含特殊补助)
• 特殊补助明细
• 综合结论

【下载报告】
点击"下载Excel报告",系统生成包含详细计算过程的Excel文件


3.5 历史记录
────────────────────────────────────────────────────────

【查看本校记录】
① 点击"历史测算记录"
② 查看本校所有测算记录
③ 点击记录查看详情

【下载历史报告】
单条下载:点击记录的"下载"按钮
批量下载:勾选多条记录 → 点击"批量下载"


================================================================================
                          四、常见问题
================================================================================

Q1: 忘记密码怎么办?
A: 联系管理员重置密码

Q2: 数据保存后能否修改?
A: 可以。选择相应年份和口径,修改后重新保存即可

Q3: 同一年份可以有多条记录吗?
A: 可以,通过不同的"测算口径"区分(如基准口径、发展口径)

Q4: 什么是"含特殊补助"和"不含特殊补助"?
A: 不含特殊补助:按教育部标准计算的基本缺口
   含特殊补助:考虑重点实验室、附属医院等特殊需求后的实际缺口

Q5: 为什么下载的Excel中某些数据为0?
A: 检查数据录入是否完整,确认该项是否确实为0

Q6: 批量下载的ZIP文件打不开?
A: 重新下载,或更换浏览器(推荐Chrome)

Q7: 可以删除历史记录吗?
A: 学校用户可删除本校记录,基建中心只能查看
   ⚠️ 删除不可恢复

Q8: 系统支持手机访问吗?
A: 支持,但建议使用电脑以获得最佳体验


================================================================================
                          五、注意事项
================================================================================

【数据录入】
✓ 面积统一使用平方米(m²),保留两位小数
✓ 人数使用整数
✓ 数据应与学校实际情况和规划文件一致
✓ 录入过程中定期保存,避免数据丢失

【系统使用】
✓ 推荐使用Chrome或Edge浏览器
✓ 使用HTTPS访问更安全
✓ 定期更换密码(建议每3个月)
✓ 使用完毕后记得退出登录

【测算结果】
✓ 缺口为正表示不足,为负表示富余
✓ 关注含/不含特殊补助两种结果的差异
✓ 数据变化后应重新测算
✓ 重要数据建议导出Excel保存

【安全提醒】
✓ 不要将账号密码告知他人
✓ 不要在公共电脑保存密码
✓ 下载的数据文件应妥善保管
✓ 发现账号异常立即报告管理员
